# Ad Management System

ASP.NET Core MVC platform for managing digital advertisements with Minimal Dashboard UI.

## Features

- 🎨 Modern Minimal Dashboard UI design
- 👥 Role-based authentication (Admin, Publisher, Advertiser)
- 📊 Real-time analytics dashboard
- 📱 Fully responsive RTL/Arabic support
- 🔐 ASP.NET Core Identity integration

## Tech Stack

- ASP.NET Core MVC (.NET 6/7/8)
- Entity Framework Core
- SQL Server
- Bootstrap 5.3.2 RTL
- Chart.js for analytics

## Local Development

1. Clone the repository
2. Update `appsettings.json` with your local database connection
3. Run migrations: `dotnet ef database update`
4. Run the application: `dotnet run`
